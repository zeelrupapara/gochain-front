## How to Take Screenshots

### Option 1: Browser Built-in (Recommended)
**Chrome/Edge:**
1. Press `F12` to open DevTools
2. Press `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac)
3. Type "screenshot" and select "Capture full size screenshot"
4. Save the image

**Firefox:**
1. Press `Shift+F2` to open Developer Console
2. Type `screenshot --fullpage`
3. Press Enter

### Option 2: Browser Extensions
- **Awesome Screenshot** (Chrome/Firefox)
- **Fireshot** (Chrome/Firefox)
- **Nimbus Screenshot** (Chrome/Firefox)

### Option 3: Online Tools
- **Screely.com** - Add browser mockup frames (looks professional)
- **Screenshot.rocks** - Create beautiful browser mockups
- **Browserframe.com** - Add browser chrome to screenshots
